HTML规范

  1. 引入 CSS 时必须指明 rel="stylesheet"

  2. 引入 CSS 和 JavaScript 时无须指明 type 属性

  3. 展现定义放置于外部 CSS 中，行为定义放置于外部 JavaScript 中

  4. 在 head 中引入页面需要的所有 CSS 资源

  5. JavaScript 应当放在页面末尾，或采用异步加载

  6. 保证 favicon 可访问

  7. 若页面欲对移动设备友好，需指定页面的 viewport

  8. 对于无需自闭合的标签，不允许自闭合

    常见无需自闭合标签有input、br、img、hr等
    
<!-- Good -->
<img src="pingan.png" alt="pingan">
<input type="text">

<!-- Not Good -->
<img src="pingan.png" alt="pingan"/>
<input type="text"/>

  9. 对 HTML5 中规定允许省略的闭合标签，不允许省略闭合标签

  10. 标签名必须使用小写字母。

  11. 标签使用必须符合标签嵌套规则

    比如 div 不得置于 p 中，tbody 必须置于 table 中

  12. 属性名必须使用小写字母，属性值必须用双引号包围(不允许使用单引号)

  13. 自定义属性建议以 xxx- 为前缀，推荐使用 data-

  14. 禁止 img 的 src 取值为空。延迟加载的图片也要增加默认的 src，避免为 img 添加不必要的 title 属性，为重要图片添加 alt 属性

  15. 添加 width 和 height 属性，以避免页面抖动

  16. 有下载需求的图片采用 img 标签实现，无下载需求的图片采用 CSS 背景图实现

  17. 有文本标题的控件必须使用 label 标签将其与其标题相关联

  18. 使用 button 元素时必须指明 type 属性值

  19. 当使用 JavaScript 进行表单提交时，如果条件允许，应使原生提交功能正常工作
